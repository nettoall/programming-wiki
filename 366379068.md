# 연산자

## 대입연산자

- a = 1;

## 비교연산자

- == : 동등 연산자 return true/false, 타입 검사 안 함

- === : 일치 연산자, 좌항과 우항이 “정확”하게 같은 때 true/false. 타입까지 검사

ex)

1 == '1' // true

1 === '1' // false

null == undefined // true

null === undefined // false

true == 1 // true

true === 1 // false

true == '1' // true

true === '1' // false

== 보다 === 사용을 추천

- != : 부정

- !== : 정확하게 같지 않다. === 참고

- \> : 좌항이 우항보다 크다

- \>= : 크거나 같다.

### 연산자 사용 예

const a = 1;

const b = '1';

const equals = a === b; // false

const equals = a == b; //true

0 == false; // true

null == undefined; // true

### 두 값 일치

!== : 타입 검사까지 수행

!= : 타입 검사 수행 안 함

# 변수 선언

- let

  - var로 정의 안한 변수는 전역 변수가 되기도 함(ES 6 이전)

  - ES6 부터는 var로 변수 선언 안하는 추세임

  - var는 함수 영역 scope 변수 선언시

  - let, const는 block 영역 scope

  - let {} 블럭 내에서만 유효

    <div class="code panel pdl" style="border-width: 1px;">

      let sum = 0; // let 선언 변수, 재선언 안 됨

        sum += i;
      }
      console.log(sum);
    }
