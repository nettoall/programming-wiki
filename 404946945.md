### useRef로 특정 DOM 선택

- ref로 DOM 선택

- 함수형 컴포넌트에서 useRef hook 함수

- 클래스형 컴포넌트에서는 콜백 함수 또는 React.createRef 함수 사용

- useRef() 를 사용해서 Ref 객체를 만들고 이 객체를 우리가 사용하고 싶은 DOM에 ref 값 설정

- .current 값은 원하는 DOM을 가르킴

- ref.current.focus 하면 원하는 위치(ref) 포커싱함

#### 변수 관리

- 컴포넌트 안에서 조회 및 수정할 수 있는 변수 관리

### 배열 렌더링

- 동적으로 배열을 렌더링 하려면 map 사용

- \<User user={user} /\> 만 정의시 key 속성을 설정하라는 오류 발생. 배열은 고유값을 설정해야 함.

- 고유값 id가 없을 경우 map(0 함수를 사용할 때 설정하는 콜백 함수의 두번째 파라미터 index를 key로 사용

  <div class="code panel pdl" style="border-width: 1px;">

  ``` syntaxhighlighter-pre
  function UserList() {
    const users = [
      {
        id:1,
        username: 'ddd',
        email: 'ddd@ddd.com'
      },
      ...
    ];
    return (
      <div>
        {users.map(user => (  // users.map() 함수 사용. user는 각 원소를 의미
          <User user={user} key={user.id} /> // key 설정 필수
        ))}
      </div>
    )
  }

  function User({user}) {
    return (
     <div>
       <b>{user.name}</b> <span>{user.email}</span> {/* 각 개별 출력 설정 */}
     </div>
    )
  }
  ```

  - key가 있을 경우, 해당 키의 위치를 지정해서 처리할 수 있으므로 ‘새로 입력된 5의 key을 가진 값을 ‘1’과 '2' 사이에 입력 또는 삭제가 가능

  ``` syntaxhighlighter-pre
        username,
        email

      // setUsers(Users.concat(user));  // concat: 새로운 배열을 생성
      
      setInputs({

        email: ''
  ```
