## 모듈

### 정의

- 함수나 변수 또는 클래스를 모아 놓은 파일

- 다른 파이썬 프로그램에서 불러와 사용할 수 있게끔 만든 파이썬 파일

### 모듈 만들기

- 예: mod1.py 파일(C:\doit에 저장)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  def add(a,b):
    return a + b
    
  def sub(a,b);
    return a - b
  ```

  </div>

  </div>

### 모듈 불러오기

1.  command line 에서 불러오기

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    cd c:\doti  // 모듈이 저장된 디렉토리로 이동)
    python
    >>> import mod1   // import는 현재 디렉토리에 있는 파일중 모듈 불러옴
    >>> print(mod1.add(3,4))
    ```

    </div>

    </div>

    - **import 모듈이름**(파일명과 동일)

2.  함수만 불러오기

    - 사용법 : **from 모듈이름 import 모듈함수명**

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      >>> from mod1 import add
      >>> add(4,2)
      ```

      </div>

      </div>

    - 여러 모듈함수 불러오기

    - 사용법 : **from 모듈이름 import 모듈함수명1, 모듈함수명2**

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      >>> from mod1 import add, sub
      >>> from mod1 import *
      ```

      </div>

      </div>

- 모듈 내에 **실행 명령어 정의했을 때** **import시 실행하지 않도록** 처리 방법

  - mod1.py 파일 저장

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    # mod1.py
    def add(a,b):
      return a + b
      
    print(add(3,4))    // print() 실행 명령어 정의 
    ```

    </div>

    </div>

  - 명령어 실행

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> import mod1
    7   // import 시 print()가 실행됨
    ```

    </div>

    </div>

  - mod1.py 파일에 다음 조건문 추가

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    if __name__ = "__main__":  // __name__ 파이썬 내부 변수
      print(add(3,4))
    ```

    </div>

    </div>

    - 자기 자신을 실행시 ‘**\_\_**name**\_\_**’ 에 ‘**\_\_main\_\_**’ 저장.

    - import 시 mod1.py의 ‘\_\_name\_\_’에 mod1이 저장됨

    - import 한 모듈이 자신이 아니므로 실행되지 않음

### 클래스,변수 선언 모듈

- 자바스크립트처럼 여러 클래스, 함수, 변수 정의 가능

- 클래스 class, 함수 def

- 선언 정의

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  # mod2.py
  PI = 3.141592

  class Math:
    def solv(self, r):
      return PI * (r **2)
      
  def add(a,b):
    return a + b
  ```

  </div>

  </div>

- 모듈 안에 클래스, 함수, 변수 마음껏 작성 가능

- 사용 예

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import mod2
  >>> print(mode2.PI)
  3.141592
  >>> a = mod2.Math()
  12.566368
  >>> print(mode2.add(mod2.PI, 4.4))
  7.541592
  ```

  </div>

  </div>

### 다른 파일에서 모듈 부르기

- 같은 디렉토리에 위치

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  # test.py
  import mode2
  result = mode2.add(3,4)
  print(result)
  ```

  </div>

  </div>

- 다른 디렉토리 모듈 부르기

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import sys   // 파이썬 제공 기본 모듈
  >>> sys.path     // 파이썬 라이브러리 설치 디렉토리 조회 명령어
  >>> sys.path.append("c:\doit\module")  // 라이브러리에 추가
  ```

  </div>

  </div>

  - sys.path : 파이썬 라이브러리 디렉토리 조회

  - sys.path.append 로 디렉토리 추가

- 파이썬 환경변수 사용(PYTHONPATH 정의)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  set PYTHONPATH=c:\doti\module
  ```
