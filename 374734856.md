# Node.js

- 오픈소스 서버 환경

- 웹 브라우저가 아닌 컴퓨터(서버)에서 자바 스크립트를 실행할 수 있도록 해줌

- 여러 플랫폼(Windows, Linux, Unix, Mac OS, etc.)에서 실행가능

- 서버에서 JavaScript를 실행

- 비동기 프로그래밍

- 구글에서 공개한 소프트웨어 V8 엔진을 기반으로 만든 자바스크립트 런타임 도구

## 활용범위

- 동적 페이지 콘텐츠 생성

- 서버에서 파일을 생성, 열고 읽고 쓰고 삭제 등을 수행

- 데이터 수집

- 데이터베이스를 조작

## NodeJs 프레임워크

- MVC, Full-stack MVC, REST API 프레임워크 세가지 유형으로 분류됨

### **Express.JS**

- 빠르고 미니멀리스트 웹 프레임워크

- 웹 애플리케이션 및 API 구축을 위해 설계

- 웹 및 모바일 애플리케이션 개발을 위한 당력한 기능 모음을 제공하는 빠르며 최소한의 유연한 MVC Node.js 프레임워크

- 기능

  - 고성능에 중점을 둔 강력한 라우팅

  - HTTP 도우미(리다이렉션, 캐싱)

  - 보기시스템 14+지원 엔진 지원

  - 높은 적용 범위 디버깅 매커니즘

  - 콘텐츠 협상

### <a href="http://Socket.io" class="external-link" rel="nofollow"><strong>Socket.io</strong></a>

- 실시간 애플리케이션 구축을 위한 빠르고 안정적인 fullstack 프레임워크

- 실시간 양방향 이벤트 기반 통신을 위해 설계

- 자동 재접속 지원, 단절 감지, 데이터 직렬화의 바이너리 지원, 멀티플레싱 및 룸 지원 , 간단하고 편리한 API

### Meteor.js

- 자바스크립트 앱 플랫폼

- 앱, 모바일, 데스크톱 애플리케이션 개발에 사용

- 기능

  - 통합 라이브 브라우저 다시 로드

  - 맞춤형 패키지 관리자

  - 데이터베이스 통합(MongoDB)

  - 반응형 템플릿

  - 프로적션 서버에서 자동 CSS 및 JS 축소

### Nest.js

- 동적이고 확장 가능한 서버측 애플리케이션을 구축하기 위한 점진적인 프레임워크

- 백엔드 생산성 향상

- TypeScript 로 작성된 프로그레시브 JavaScript 사용

- 기능

  - TypeScript 활용

  - 강력한 CLI 보유

  - 공통 기술과의 손쉬운 통합을 용이하게 하는 여러 개의 특정 모듈 지원

  - 모놀리식 및 마이크로서비스용으로 설계됨

  - 쉬운 단위 테스트

### Koa.js

- HTTP 미들웨어 프레임워크

- 웹 애플리케이션 및 API를 빌드하는데 사용

- 기능

  - 비동기 및 공통 기능 사용

  - 컨텍스트 객체 사용

  - 모던하고 가벼움

  - 오류 처리에 좋음

### Sails.js

- 실시간 MVC 프레임워크

- 맞춤형 엔터프라이즈급 Node.js 애플리케이션 생성을 간소화

- Node.js, Express 및 Socket.io를 기반으로 하는 프레임워크

- Connect 미들웨어와 호환

- 앱에서 실시간 기능 구축 가능

- 기능

  - JavaSript로 전체 앱 빌드

  - 여러 데이터베이스 지원

  - 데이터 모델링을 보다 실용적으로 만듬

  - 자동 생성 REST API

  - 추가 코드없이 Websocket 지원

  - 프론트 엔드 불가지론?

  - 선언적이로 재사용 가능한 보안 정책

### Mocah.js

- 자바스크립트 테스트 프레임워크

- 비동기 테스트를 간단하고 재미있게 만드는 기능이 풀부

- 기능

  - BDD 및 TDD 모두 지원

  - 여러 설치 옵션

  - 여러 어셜션 라이브러리 지원

  - 비동기 테스트 지원

  - 여러 우베 브라우저와 호환

### Restify.js

- Restful 서비스 개발에 최적화된 웹 서비스 프레임워크

- Netflix, Pnterest 및 Napster 등 사용중

- 기능

  - Node.js 배포 정원 켜기

  - 더 나은 디버깅 기능으로 구축

### Adonis.js

- 백엔드 프레임워크

- 기능

  - 웹 앱 또는 API 서버를 생성할 수 있는 에코시스템 제공

  - 경로 그룹 및 하위 도메인 기반 라우팅을 지원하는 기능이 풍부한 라우팅 레이어 제공

  - 컨트롤러는 전용 컨트롤러 파일에 대한 인라인 경로 처리기 제거

  - 파입 업로드 관리 지원

  - 스키마 기반 유효성 검사기로 런타임 유효성 검사 및 정적 유형 안정성 제공

  - 서버 렌더링 웹 앱을 만들기 위한 템플릿 엔진

### Loopback 4

- API 및 마이크로서비스 구축을 위한 프레임워크

- Express를 기반으로 하는 프레임워크

- 복잡한 통합이 필요한 최신 API 애플리케이션의 개발 단순화

- 기능

  - 더 빠르고 가벼우며 강력함

  - 단일 명령으로 실제 API 생성

  - OpenAPI로 데이터 및 엔드포인트 정의
