1.  String 클래스는 char\[\] + 메소드 결합한 것

    - String class = char\[\] + method

2.  String 클래스는 read only (재설정시 새로운 메모리 주소를 가리킴)

3.  주요 메서드

    - charAt(int index) : 문자열에서 해당 위치(index)에 있는 문자 반환

    - length()

    - substring(int from, int to)

    - equals(Object obj)

    - char\[\] toCharArray() : 문자열을 문자배열로 변환하여 반환
