# 개요

## buildscript

- gradle로 task를 수행할 때 사용되는 설정

- buildscript 내에 정의된 dependencies는 task를 사용할 때 사용되는 라이브러리

- buildscript 밖에서 정의된 dependencies는 소스를 컴파일 할 때 사용

- buildscript는 소스 코드 컴파일 과 같은 빌드 작업을 시작하기 전에 빌드 시스템 준비 단계에서 제일 먼저 실행

- gradle 자체를 위한 것, 빌드를 수행하는 방법에 대한 변경을 명시

- spring boot 버전 정보, repository 정보, dependency 모듈을 지정

- **빌드 스크립트에서 외부 라이브러리를 사용해야 하는 경우** 빌드 스크립트 자체에서 스크립트의 클래스 경로에 추가할 수 있음. 빌드 스크립트 클래스 경로를 선언하는 블록을 전달하는 buildscript() 메소드를 사용하여 이 작업을 수행
