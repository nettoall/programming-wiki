### format 심화

- format 코드를 숫자와 같이 사용하면 정렬이나 공백을 나타낼 수 있음

- 정렬, 공백

  - “%10s” % “hi” : 전체 길이 10 문자열 오른쪽 정렬, 나머지 공간 공백 처리

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> "%10s" % "hi"
    '        hi'  # 전체 길이 10문자열, hi 앞 공백 8자리
    >>> "%-10sJane" % "hi"
    'hi        Jane'
    # hi가 Jane보다 왼쪽에 공백 8자리. hi부터 jane까지 10자리
    ```

    </div>

    </div>

- 소숫점 표현

  - %0.소수점포인트 숫자 개수 (%0.4f : 소수점 이하 4자리수)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> "%0.4f" % 3.42134234. # 소수자리 4자리까지 출력
    3.4213
    >>> "%10.4f" % 3.42134234
        3.4213
    ```

    </div>

    </div>

### format함수 사용(.format())

- 대체

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
>>> "I eat {0} apples".format(3)
'I eat 3 apples'
>>> "I eat {0} apples".format("five")
'I eat five apples'
>>> number = 3
>>> "I eat {0} apples".format(number)
'I eat 3 apples'
>>> number = 10
>>> day = "three"
# 2개 이상의 값 넣기
>>> "I ate {0} apples. so I was sick for {1} days".format(number,day)
'I ate 10 apples. so I was sick for three days.'
# 이름으로 넣기
>>> "I ate {number} apples. so I was sick for {day} days".format(number=10, day=3)
'I ate 10 apples. so I was sick for 3 days'
# 인덱스와 이름 혼용하기
>>> "I ate {0} apples. so I was sick for {day} days".format(10, day=3)
'I ate 10 apples. so I was sick for 3 days.'
```

</div>

</div>

- 정렬

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> "{0:<10}".format("hi")
  # 출력결과 'hi          '
  >>> "{0:>10}".format("hi")
  # 출력결과 '          hi'
  >>> "{0:^10}".format("hi")
  # 출력결과 '     hi     '
  ```

  </div>

  </div>

- 공백 채우기

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> "{0:=^10}".format("hi")
  # 출력 결과 '====hi===='
  >>> "{0:!<10}".format("hi")
  # 출력 결과 'hi!!!!!!!!'
  ```

  </div>

  </div>

- format 구조

  - {:} : 앞에 대치할 인덱스 or name, 뒤에 format 자리수, 정렬 형식

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> y = 3.42134234
    >>> "{0:0.4f}".format(y)
    '3.4213'
    >>> "{0:10.4f}".format(y)
    '         3.4213'  # 공백 9자리
    >>> "{{and}}".format()  # 문자 표현하기 
    # 출력 결과 : {and}
    ```

    </div>

    </div>

- 파이썬 3.6 이상부터 **f 문자열 formatting 기능 추가**됨

  <div class="code panel pdl" style="border-width: 1px;">


    </div>

    </div>

  - 딕셔너리 f 문자열 formatting 기능 예제(딕셔너리는 key, value 자료형)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">


### 교집합, 합집합, 차집합 구하기

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> s1 = set([1,2,3,4,5,6])
  >>> s2 = set([4,5,6,7,8,9])
  >>> s1 & s2 # 교집합
  {4,5,6}
  >>> s1.intersection(s2) # 교집합
  {4,5,6}
  >>> s1|s2 # 합집합
  {1,2,3,4,5,6,7,8,9}
  >>> s1.union(s2)
  {1,2,3,4,5,6,7,8,9}
  >>> s1 - s2 # 차집합
  ```
