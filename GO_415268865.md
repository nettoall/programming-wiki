# 개요

- GO 공식 사이트 : <a href="https://go/dev" class="external-link" rel="nofollow">https://go/dev</a> (<a href="https://golang.org" class="external-link" data-card-appearance="inline" rel="nofollow">https://golang.org</a> )

- 2007년 구글에서 개발을 시작, 2012년 GO 버전 1.0 완성

- <a href="http://golang.site" class="external-link" data-card-appearance="inline" rel="nofollow">http://golang.site</a>

# 특징

- 전통적인 컴파일, 링크 모델을 따르는 범용 프로그래밍 언어

- 일차적으로 시스템 프로그래밍을 위해 개발

- C++, Java, Python의 장점들을 뽑아 만들어짐

- C++와 같이 컴파일러를 통해 컴파일, 정적 타입(Statically Typed) 의 언어

- Java와 같이 Garbage Collectoin 기능 제공

- 단순하고 간결한 프로그래밍 언어 지향, 25개의 키워드 만으로 프로그래밍 가능

- Go는 Communicating Sequential Processes(CSP) 스타일의 Concurrent 프로그래밍 지원

# 환경 구성

## 설치

### OS 별 설치

#### 윈도우

- msi로 Go 언어 설치

- 기본 디렉토리는 C:\Program Files\Go 에 설치됨

#### WSL2

- wget <a href="https://golang.org/dl/go1.21.4.linux-amd64.tar.gz" class="external-link" rel="nofollow">https://golang.org/dl/go1.21.4.linux-amd64.tar.gz</a> (다운로드)

- suto tar -xvf go1.21.4.linux-amd64.tar.gz (압축 해제)

- sudo mv go /usr/local (공용 bin 디렉토리로 go 실행프로그램 이동)

- path 설정

  - ~/.zshrc 에 추가 (shell zsh)

  - export GOROOT=/usr/local/go

  - export GOPATH=\$HOME/go

  - export PATH=\$GOPATH/bin:\$GOROOT/bin:\$PATH

### GO 환경변수

- go env 명령으로 환경변수 확인

- Go는 두 개의 중요한 환경변수 사용 (GOPATH, GOROOT)

  - GOROOT : Go가 설치된 디렉토리 ( ex) C:\Program Files\Go)

  - Go의 실행파일은 GOROOT/bin 에 위치

  - Go의 표준 패키지는 GOROOT/src 폴더에 설치됨

  - GOPATH : Go 프로젝트의 홈 디렉토리 역할

  - GOPATH/bin 디렉토리는 3rd party 실행 파일 및 사용자 프로젝트의 go 실행 파일 저장

  - go install 실행시 GOPATH/bin에 저장

### VSC와 Go 사용

- Extension “Go” 설치

- 작업 폴더 생성 ( ex : GoApp/MyProject)

  - bin, pkg, src 3개 서브 디렉토리 생성 (go 표준인듯)

- VSC 에서 방금 생성한 작업 폴더 열기

- gopls 설치에 관련된 메시지가 나오면 설치 (gopls 는 Go 문법 체크 및 인텔리센스 기능 제공)

## 프로그램

### 변수

- 변수 선언 형식

  - var 변수명 변수타입 (ex: var a int)

- 변수 초기값 할당

  - var 변수명 변수타입 = 초기값 (ex : var f float32 = 11. ) 11. 은 11.0 을 의미

- 변수 값 할당

  - 변수 선언 후 값 할당 가능

  - ex: a = 10 , f = 12.0

- 선언된 변수가 Go 프로그램 내에서 사용되지 않으면 에러 발생(삭제 필요)

- 동일한 타입의 변수가 여러개 있을 때 선언

  - var i, j, k int

- 복수 변수들이 선언된 상황에서 초기값 지정

  - var i, j, k int = 1, 2, 3

- Go 에서 변수 타입 선언 안 했을 시 값을 보고 추론하는 기능 제공

  - var i = 1

  - var s = “Hi”

- Short Assignment Statement (:=) 변수를 선언하는 또 다른 방식

  - i := 1 (var 생략 함수 내에서만 사용)

### 데이터 타입

- Go는 다음과 같은 기본 데이터 타입을 갖는다.

  - bool : boolean type

  - string : 문자열 (한번 생성되면 수정될 수 없는 Immutable 타입)

  - int int8 int16 int32 int64 : 정수형 타입

  - uint uint8 uint16 uint32 uint64 uintptr : 정수형 타입

  - float32 float64 complex64 complex128 : Float 및 복소수 타입

  - byte : uint8과 유사, 바이트 코드에 사용

  - rune : int32와 동일, 유니코드 코드 포인트에 사용

#### 문자열

- '' 혹은 ““ 를 사용하여 표현

- ‘' : Raw String Literal. '’ 로 둘러싸인 문자열은 별도로 해석되지 않고 Raw String 그대로의 값을 갖음

  - ‘\n’ 이 있을 경우 New Line 으로 해석되지 않고 \n 로 사용

  - 복수 라인의 문자열을 표현할 때 자주 사용

- ““ : Interpreted String Literal

  - 복수 라인에 걸쳐 쓸수 없음

  - ““ 안의 Esacpe 문자열은 특별한 의미로 해석됨 “\n” 은 New Line 으로 해석됨

  - 문자열을 여러 라인에 걸쳐 쓰기 위해 + 연산자 사용

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  package main

  import "fmt"

  func main() {
    //Raw String Literal
    rawLiteral := '아리랑\n
  아리랑\n
    아라리요'
    
    // Interpreted String Literal
    interLiteral := "아리랑아리랑\n아라리요"
    // 여러 라인에 걸쳐 사용하려면 + 사용
    
    fmt.println(rawLiteral)
    fmt.println()
    fmt.println(interLiteral)
  }
  ```

  </div>

  </div>

#### 데이타 타입 변환

- 다른 타입으로 변환하기 위해서 T(v)와 같이 표현

  - float32(100) : 정수 100 을 float32 타입으로 변환

  - \[\]byte(“ABC”) : 문자열을 바이트 배열로 변환 (배열 표시\[\] 는 데이타 타입 앞에 표시)

  - 타입간 변환은 명시적으로 지정

  - 예제

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    func main() {
      var i int = 100
      var u uint = uint(i)
      var f float32 = float32(i)
      println(f, u)
      
      str := "ABC"
      bytes := []byte(str)
      str2 := string(bytes)
      println(bytes, str2)
    }
    ```

### 조건문

#### if문

- 조건식을 () 로 둘러 싸지 않아도 됨

- 조건 블럭 시작 brace ({}) 를 if문과 같은 라인에 두어야 함

- if 문의 조건식은 반드시 boolean 식으로 표현

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  if k == 1 {       // { } 은 if과 같은 라인
    println("One")
  } else if k == 2 {   // 같은 라인
    println("Two")
  } else {           // 같은 라인
    println("Other")
  }
  ```

  </div>

  </div>

- if 문 안에서만 사용할 수 있는 변수를 if 바로 전에 선언

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  if val := i * 2; val < max {
    println(val)
  }

  val++  // 오류 발생
  ```

  </div>

  </div>

  - 조건식(val \< max) 이 사용되기 전 정의된 구문은 if 문 블럭 안에서만 사용 가능

#### Switch문

- 다수의 조건식 체크

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  package main

  func main() {
    var name string
    var category = 1
    
    switch category {
    case 1:
      name = "Paper Book"
    case 2:
      name = "eBook"
    case 3, 4:    // 여러 case 동시 정의
      name = "Blog"
    default:
      name = "Other"
    }
    println(name)
    
    //Expression을 사용한 경우 
    switch x := category << 2; x - 1{
    
    }
  }
  ```

  </div>

  </div>

- Go의 특별한 Switch 용법

  - Switche 뒤에 Expression이 없을 수 있음. 이 경우 Go는 Switch expressoin을 true로 생각하고 첫번째 case 문으로 이동하여 검사

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    func grade(score int) {
      switch {
      case score >= 90:
        println("A")
      case score >= 80:
        println("B")
      case score >= 70:
        println("C")
      case score >= 60:
        println("D")
      default: 
        println("No Hope")
      }
    }
    ```

    </div>

    </div>

  - case 문에 expression을 쓸 수 있음. Go는 Case 문에 복잡한 expression을 가질 수 있음

  - No default fall through : 다른 언어의 case 문은 break를 쓰지 않는 한 다음 Case로 이동하지만, Go는 다음 Case로 가지 않는다.

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    package main

    import "fmt"

    func main() {
      check(2)
    }

    func check(val int) {
      switch val {
      case 1:
        fmt.println("1 이하")
        fallthrough          // fallthrough 사용하면 다음 case 문으로 이동
      case 2:
        fmt.println("2이하")
        fallthrough
      case 3:
        fmt.println("3이하")
        fallthrough
      default:
        fmt.println("default 도달")
      }
    }
    ```

    </div>

    </div>

  - Type switch : Go는 변수의 Type 에 따라 case로 분기할 수 있다.

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    switch v.(type) {
      case int:
        println("int")
      case bool:
        println("bool")
      case string:
        println("string")
      default:
        println("unknown")
    }
    ```
