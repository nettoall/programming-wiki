- 최대값, 최소값을 쉽게 추출

- 일종의 반정렬 상태 유지

  - 부모노도의 키값이 자식 노드의 키값보다 항상 큰( 또는 작은) 이진트리

  - 중복값 허용

힙종류

- 최대힙 : 부모노드 키값 \>= 자식노드키값

- 최소힙 : 부모노드키값 \<= 자식노드키값

힙 구현하기

- 배열로 구현

- 첫번째 인덱스 0로 사용하지 않음

부모와 자식간의 관계

- 왼쪽 자식의 인덱스 = (부모 인덱스 ) \*2

- 오른쪽 자식의 인덱스 = (부모 인덱스) \*2 + 1

- 부모의 인덱스 = (자식의 인덱스) /2

삽입

- 새로운 노드는 힙의 마지막 노드에 이어서 삽입

- 새로운 노드를 부모노드들과 교환하여 힙의 성질을 만족시킴
