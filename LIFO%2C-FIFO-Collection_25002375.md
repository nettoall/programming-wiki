# LIFO(Last In, First Out)

- 대표적인 collection 사용 예 ; Stack, JVM 스택 메모리

## Stack

- 메소드

<div class="table-wrap">

|               |              |                                          |
|---------------|--------------|------------------------------------------|
| **리턴 타입** | **메소드**   | **설명**                                 |
| E             | push(E item) | 스택에 객체 입력                         |
| E             | peek()       | 스택에서 맨 위 객체를 가져온다 제거 안함 |
| E             | pop()        | 스택에서 맨 위 객체를 가져오면서 제거    |

</div>

- 사용 예\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Stack<E> stack = new Stack<E>();
  ```

  </div>

  </div>

<!-- -->

- 예제\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  public class Coin {

      int amt;
      
      public Coin(int amt) {
          this.amt = amt;
      }
  }
  ```

  </div>

  </div>

  \
  \

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  public class StackExample {

      public static void main(String[] args) {
          Stack<Coin> coinBox = new Stack<Coin>();        
          coinBox.push(new Coin(100));
          coinBox.push(new Coin(200));
          coinBox.push(new Coin(300));
          
          while(!coinBox.isEmpty()) {
              Coin coin = coinBox.pop();
              System.out.println(coin.amt);
          }

      }
  }
  ```

  </div>

# FIFO(First in First Out)

## Queue

- 사용 예 : 스레드 풀 (ExecutorService)

- 메소드

<div class="table-wrap">

|               |            |                                   |
|---------------|------------|-----------------------------------|
| **리턴 타입** | **메소드** | **설명**                          |
| boolean       | offer(E e) | 주어진 객체 입력                  |
| E             | peek()     | 객체 하나를 가져오는데 삭제 안 함 |
| E             | poll()     | 객체 하나를 가져오면 삭제         |

</div>

- LinkedList는 Queue 인터페이스 구현

- 기본 정의\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Queue<E> queue = new LinkedList<E>();
  ```

  </div>

  </div>

<!-- -->

- 사용 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  public class Message {
      String address;
      String name;
      
      public Message(String address, String name) {
          this.address = address;
          this.name = name;
      }

  }
  ```

  </div>

  </div>

  \

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  public class QueueExample {
      public static void main(String[] args) {
          Queue<Message> messageQueue = new LinkedList<Message>();
          messageQueue.offer(new Message("Send Mail","홍길동"));
          messageQueue.offer(new Message("Send Kakao","홍두깨"));
          messageQueue.offer(new Message("Send TokTok","이길동"));
          
          while(!messageQueue.isEmpty()) {
              Message message = messageQueue.poll();
              System.out.println(message.address + ":" + message.name);
          }
      }
  }
  ```

  </div>
