- range()

  - 숫자 리스트를 자동으로 만들어 주는 함수

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre

    marks = [90,25,67,45,80]
    for number in range(len[marks])
      if marks[number] < 60: continue
        print("%d번 학생 축하" % (number + 1))
    ```

    </div>

    </div>

  - range를 쓰지 않을 때에는 number를 초기화해야 한다.

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre

    number = 0 # 초기화
    for mark in marks:
      number = number + 1
      if mark < 10: continue
    ```

### 리스트 내포하기

- 일반 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> a = [1,2,3,4]
  >>> result = []
  >>> for num in a:
    result.append(num * 3)
  >>> print(result)
  [3, 6, 9, 12]
  ```

  </div>

  </div>

- 리스트 내포한 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> a = [1,2,3,4]
  >>> result = [num * 3 for num in a]  # 리스트 내포한 for 문
  >>> print(result)
  [3, 6, 9, 12]
  # 짝수만 처리시
  >>> result = [num * 3 for num in a if num % 2 == 0]
  ```

  </div>

  </div>

- 리스트 반복 사용시

  - for문을 두개 이상 사용 가능

  - \[표현식 for 항목1 in 반복가능객체1 if 조건1 for 항목2 in 반복가능 객체2 if 조건2 for 항목3 in 반복가능객체3 if 조건3\]

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    result = [x * y for x in range(2,10)
                    for y in range(1,10) ]
    ```
