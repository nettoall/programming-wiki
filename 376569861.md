# Spring 3.1에 추가된 새로운 기능

- 강화된 Java Code를 이용한 Bean 설정

  - XML을 전혀 사용하지 않고 Spring Application 작성 가능

  - 다양한 Annotaion 추가

  - XML의 전용 Custom Tag를 대체할 수 있는 @Enable 로 시작되는 전용 Annotation 제공

- Runtime 환경 추상화

  - Runtime 환경 정보를 추상화한 환경 Object가 Container를 통해 제공

  - Runtime 환경에 따라 달라지는 Bean 설정을 효과적으로 관리하는 Profile

  - PropertySource

- JPA 지원 확장과 Hibernate 4 지원

- 새로운 DispatcherServlet 전략과 FlashMap

  - MVC 기능 확장 수월

  - Post/Redirect/Get 패턴에 사용할 수 있는 FalshMap 기능 추가

- Cache 추상화

  - Application Bean 에 Cache 기능 적용

  - Map을 이용한 간단한 Cahce 구현부터 Ehcache를 이용한 고급 캐시 기술 지원
