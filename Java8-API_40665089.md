# String

## join

- 주어진 delimiter를 여러개의 String을 하나의 String으로 변환하면서 구분자로 넣는다.\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  String.join(":","foobar","foo","bar"); // foobar:foo:bar
  ```

  </div>

  </div>

## chars

- String의 모든 character에 대해 stream을 만든다.\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  "foobar:foo:bar"
    .chars()
    .distinct()
    .mapToObj(c -> String.valueOf((char)c))
    .stored()
    .collect(Collectors.joining());
    // :abfor
  ```

  </div>

  </div>

## regex

- regex 패턴도 stream을 사용하여 String에 대해 각각의 character로 나누는 대신 패턴으로 String을 분리하고 새로운 Stream을 생성할 수 있다.\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Pattern.compile(":")
    .splitAsStream("foobar:foo:bar")  //foobar, foo, bar
    .filter(s -> s.contains("bar")) // foobar, bar
    .stored()
    .collect(Collectors.joining(":")); // bar:foobar
  ```

  </div>

  </div>

- regex 패턴은 predicates로 변환할 수 있다. 이러한 predicates는 String stream의 filter로 사용한다.\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Pattern pattern = Pattern.compile(".*@gmail\\.com");
  Stream.of("bob@gmail.com","alice@hotmail.com")
    .filter(pattern.asPredicate())
    .count(); // 1
  ```

  </div>

  </div>

- 예제의 패턴은 “@gmail.com”으로 끝나는 String을 매칭하고 Java8의 Predicate를 사용하여 이메일 주소의 stream을 필터링하였다.

# Crunching Numbers

<a href="https://winterbe.com/posts/2015/03/25/java8-examples-string-number-math-files/" class="external-link" rel="nofollow">https://winterbe.com/posts/2015/03/25/java8-examples-string-number-math-files/</a>

- Java8 은 unsigned 숫자에 대해 지원한다.

- int는 2147483647부터 -2147483648 까지 지원한다.\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  System.out.println(Integer.MAX_VALUE);      //2147483647
  System.out.println(Integer.MIN_VALUE + 1);  // -2147483648
  ```

  </div>

  </div>

- java8은 unsigned int를 파싱하도록 지원한다.\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  long maxUnsignedInt = (1l << 32) - 1;
  String string = String.valueOf(maxUnsignedInt);
  int unsignedInt = Integer.parseUnsignedInt(String, 10);
  String string2 = Integer.toUnsignedString(unsignedInt, 10);
  ```

  </div>
