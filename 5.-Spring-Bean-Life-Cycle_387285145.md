- Spring Framework의 Context 에서 Bean 생성 후 코드를 실행하려는 경우 사용자 정의 init() 메소드에서 구현할 수 있음

- 또는 Bean이 destroy 되기 전에 어떤 코드를 실행하려 한다면 destroy() 메소드 안에 구현 가능하며 이러한 메소드 들을 Spring Bean life Cycle method 라 함

- Spring Bean Life Cycle & Methods : Application Started → Spring Container Started → Bean Constructed & Dependency Injected → Custom init() Called (afterPropertiesSet(), @PostConstruct ) → Other Method Called → Custom destroy() Called (destroy(), @PreDestroy )

- Bean Life Cycle은 Spring Container 에 의해 관리

## Bean Life Cycle

- bean이 인스턴스화되는 시기와 방법, bean이 살아 있을 때 수행하는 작업 및 파괴되는 시기와 방법에 대한 것

## Spring Bean Life Cycle Methods

- Spring container 가 Bean 을 생성할 때 모든 Bean에게 기본적으로 제공하는 두가지 메소드

  - public void init()

  - public void destroy()

- method 이름은 변경 가능하지만 메서드 시그니처는 동일 해야 함

- init() 메서드는 bean 생성 후 객체를 요청하기 전에 호출됨

- destroy() 메서드는 bean이 소멸되기 직전에 호출됨

## Life Cycle Method를 구현한 목적

- init(): 일부 구성 정보 로드, 데이터베이스 연결 초기화 등 init() 에 코드 구현

- destroy() : 데이터 베이스 연결 닫기, 리소스 반환 및 정리 등 destroy() 메소드에 코드 구현

## Spring Bean Life Cycle Method를 설정하는 방법

1.  XML 사용

2.  Spring interface 사용

    - InitializingBean과 DisposalBean 상속받아 클래스 구현

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      public class Employee implements InitializingBean, DisposableBean {
        private Long id;
        private String name;
        
        // init 관련 메소드 정의
        public void afterPropertiesSet() throws Exception {
          System.out.println("Inside afterPropertiesSet() method");
        }
        
        // destroy 관련 메소드 정의
        public void destroy() throws Exception {
          System.out.println("Inside destroy() method");
        }
      }
      ```

      </div>

      </div>

    - config.xml 에 bean 등록

3.  Annotation 사용

    - pojo class에 @PostConstruct, @PreDestroy annotation 정의

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      public class Perso {
        private String name;
        private Integer age;
        
        public Person() {
          super();
        }
        
        public String getName() {
          return name;
        }
        
        public void setName(String name){
          this.name = name;
        }
        
        public Integer getAge(){
          return age;
        }
        
        public void setAge(Integer age){
          this.age = age;
        }
        
        @PostConstruct
        public void initMethod(){
          System.out.println("Inside init method");
        }
        
        @PreDestroy
        public void destroyMethod() {
          System.out.println("Inside destroy method");
        }
        
      }
      ```
