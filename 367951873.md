- 1990년 암스테르담의 귀도 반 로섬이 개발한 인터프리터 언어

# 특징

- 인간다운 언어 - 사람의 사고 체계와 닮아 있다.

- 문법이 쉽다.

- 간결하다 - 공동작업과 유지보수가 쉽고 편하다.

- 강력하다

- 개발속도가 빠르다

- 시스템 유틸리티

- GUI 프로그래밍(Tkinter GUI 도구)

- 수치연산

- 데이터 베이스 프로그래밍

- 데이터 분석, 사물 인터넷
