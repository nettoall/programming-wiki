# 정의

- `기본 인증은 사용자가 리소스에 액세스할 수 있도록 HTTP [Authorization] 헤더를 사용하여 요청의 일부로 사용자 이름과 암호를 전달하여 인증을 제공하는 방법입니다.`

- `이 유형의 인증에서 자격 증명은 쉽게 되돌릴 수 있고 보안되지 않는 Base64 인코딩 알고리즘을 사용하여 약하게 인코딩됩니다.`

## 예제

- gradle

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  plugins {
      id 'java'
      id 'org.springframework.boot' version '2.7.11'
      id 'io.spring.dependency-management' version '1.0.15.RELEASE'
  }

  group = 'com.nettoall'
  version = '0.0.1-SNAPSHOT'
  sourceCompatibility = '17'

  repositories {
      mavenCentral()
  }

  dependencies {
      implementation 'org.springframework.boot:spring-boot-starter-security'
      implementation 'org.springframework.boot:spring-boot-starter-web'
      testImplementation 'org.springframework.boot:spring-boot-starter-test'
      testImplementation 'org.springframework.security:spring-security-test'
  }

  tasks.named('test') {
      useJUnitPlatform()
  }
  ```

  </div>

  </div>

- 기본 설정 후 spring boot 실행시 username : “user', password : spring boot console에 출력된 패스워드 입력 후 로그인 가능

- application.yml 에 로그인 사용자 정보와 server port 정보 설정

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  spring:
    security:
      user:
        name: test
        password: test123
  server:
    port: 8083
  ```

  </div>

  </div>

- WebSecurityConfigurerAdapter를 상속받아 configuration 구현

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  @Configuration
  public class WebSecurityConfiguration  extends WebSecurityConfigurerAdapter {
      @Override
      protected void configure(HttpSecurity http) throws Exception {

          http.authorizeRequests()
                  .anyRequest().authenticated()
                  .and()
                  .httpBasic();
      }
  }
  ```

  </div>

  </div>

  - WebSecurityConfigurerAdapter 는 deprecated 되어서 최신 버전에 맞는 구현방법 알아봐야 함

- Sample 용 Rest Controller 구현

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  @RestController
  public class HelloController {

      @GetMapping("/hello")
      public String sayHello(@RequestParam(required = false, defaultValue = "Sample user") String name) {
          return "Hello " + name;
      }
  }
  ```

  </div>

  </div>

- Spring Boot 실행 후 웹 브라우저에서 <a href="http://localhost:8083/hello?name=User" class="external-link" rel="nofollow"><u>http://localhost:8083/hello?name=User</u></a> 입력하면 login 창이 나옴

- password 는 header 정보의 Authorization 항목에 base64로 암호화가 되어 전달됨 : 보안에 취약함
