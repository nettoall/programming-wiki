## pickle

- 객체의 형태를 유지하면서 파일에 저장 및 불러오기 (객체의 형태를 유지하면서라는 의미가…)

- 사용예

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import pickle
  >>> f = open("test.txt", 'wb')
  >>> data = {1: 'python', 2: 'you need'}
  >>> pickle.dump(data, f)  # 파일에 data 저장, 자바의 serialize와 유사
  >>> f .close()
  ```

  </div>

  </div>

- 저장한 데이터 불러오기 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import pickle
  >>> f = open("text.txt", 'rb')
  >>> data = pickle.load(f)
  >>> print(data)
  {1: 'python', 2: 'you need'}
  ```

  </div>

  </div>

## tempfile

- 파일을 임시로 만들어서 사용

- 사용 예제

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import tempfile
  >>> filename = tempfile.mktemp()  # 중복되지 않는 임시 파일의 이름을 무작위로 만듬
  >>> filename
  'c:\WINDOWS\TEMP\~2733129-0'
  >>> f = tempfile.TemporaryFile() # 임시 저장공간으로 사용할 바이너리 스기 모드로 파일 객체 반환,
  ```

## time

- 시간과 관련된 모듈

### time.time

- 현재 시간을 실수 형태로 돌려주는 함수

- 사용예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import time
  >>> time.time()
  939283829.992382382
  ```

  </div>

  </div>

### time.localtime

- time.time()이 돌려준 실수값을 사용해서 년,월,일,시,분,초로 변환

- 사용예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import time
  >>> time.localtime(time.time())
  time.struct_time(tm_year=2023, tm_mon=5, ...)
  ```

  </div>

  </div>

### time.asctime

- time.localtime()에 의해서 반환된 튜플형태의 값을 인수로 받아 날짜와 시간을 알아보기 쉬운 형태로 변환

- 사용예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> time.asctime(time.localtime(time.time()))
  'Sat Apr 28 20:50:20 2001'
  ```

  </div>

  </div>

### time.ctime

- 현재 시간을 알아보기 쉬운 형태로 변환

  <div class="codeContent panelContent pdl">


### time.strftime

- 시간에 관계된 것을 세밀하게 표현하는 포맷코드 제공

- 사용법 : **time.strftime('출력할 형식 포맷 코드', time.localtime(time.time()))**

- 포맷 코드

<div class="table-wrap">

|          |                                     |                   |
|----------|-------------------------------------|-------------------|
| **코드** | **설명**                            | **예**            |
| %a       | 요일 줄임말                         | Mon               |
| %A       | 요일/                               | Monday            |
| %b       | 달 줄임말                           | Jan               |
| %B       | 달                                  | January           |
| %c       | 날짜와 시간을 출력                  | 06/01/01 17:22:21 |
| %d       | 날(day)                             | \[01,31\]         |
| %H       | 시간: 24시간                        | \[00,23\]         |
| %l       | 시간:12시간                         | \[01,12\]         |
| %j       | 1년 중 누적 날짜                    | \[001,366\]       |
| %m       | 달                                  | \[01,12\]         |
| %M       | 분                                  | \[01,59\]         |
| %p       | AM or PM                            | AM                |
| %S       | 초                                  | \[00,59           |
| %U       | 1년 중 누적 주: 일요일을 시작으로   | \[00,53\]         |
| %w       | 숫자로 된 요일                      | \[0(일요일),6\]   |
| %W       | 1년 중 누적 주-월요일을 시작으로    | \[00,53\]         |
| %x       | 현재 설정된 지역에 기반한 날짜 출력 | 06/01/01          |
| %X       | 현재 설정된 지역에 기반한 시간 출력 | 17:22:21          |
| %Y       | 연도 출력                           | 2001              |
| %Z       | 시간대 출력                         | 대한민국 표준시   |
| %%       | 문자                                | %                 |
| %y       | 세기 부분을 제외한 연도 출력        | 01                |

</div>

- 사용예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> imprt time
  >>> time.strftime('%x', time.localtime(time.time()))
  '05/01/01'
  >>> time.strftime('%c', imte.locaimte(time.time()))
  '05/01//01 17:22:21'
  ```

  </div>

  </div>
