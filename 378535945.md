## 내장함수

- 파이썬 내장함수

- import 정의 필요 없음

### abs

- 숫자의 절대값 반환

- 사용법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> abs(3)
  3
  >>> abs(-3)
  3
  >>> abs(-1.2)
  1.2
  ```

  </div>

  </div>

### all

- 반복가능한 자료형이면 true, 거짓이 하나라도 있으면 false

- 사용법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> all([1, 2, 3])
  True
  >>> all([1, 2, 3, 0]) # 0 은 거짓
  0
  ```

  </div>

  </div>

### filter

- filter(func,x) : func 함수 이름, x 함수에 들어갈 반복가능한 자료형

- 예제

  - positive.py 정의(자료형에 있는 인자들 중 양수인 것만 추려내기)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    #positive.py
    def positove(l):
      result = []
      for i in l:
        if i > 0:
          result.append(i)
      return result
    print(positive([1, -3, 2, 0, -5, 6]))
    ```

    </div>

    </div>

  - filter.py 정의(filter 명령어로 양수인 것만 추려내기) - 앞 positive.py 보다 간단함

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    #fileter1.py
    def positive(x):
      return x > 0

    print(list(filter(positive, [1, -3, 2, 0, -5, 6])))
    ```

    </div>

    </div>

  - lambda로 정의

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> list(filter(lambda x: x > 0, [1, -3, 2, 0, -5, 6]))
    ```

    </div>

    </div>

### map

- map\[f, iterable) : 함수나 반복가능한 자료형을 받아 자료형의 각 요소를 함수 f가 수행한 결과를 묶어서 반환

- 예제 - 인자로 받은 값을 2배로 돌려주는 함수

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  # two_times.py
  def two_times(numberList):
    result = []
    for number in numberList:
      result.append(number*2)
    return result
    
  result = two_times([1, 2, 3, 4])
  print(result)
  ```

  </div>

  </div>

- map 사용 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre

  ...
  >>> list(mape(two_time, [1, 2, 3, 4]))
  [2, 4, 6, 8]
  ```

  <div class="codeContent panelContent pdl">
