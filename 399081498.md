# Byte I/O

## InputStream

### 종류

- FileInputStream

- DataInputStream

- BufferedInputStream

- ByteArrayInputStream :바이크 배열을 차례대로 읽기 위한 클래스

### 공통 method

- int read(): 1byte 를 읽고 읽은 byte를 리턴

  - 1byte를 읽고 int 타입(4byte) 로 리턴(끝의 1byte에만 값이 들어 있음)

- int read(byte\[\] b) : 읽은 byte를 byte 배열 b에 저장, 읽은 byte 수 리턴

  - 지정된 바이트 배열 크기보다 읽는 입력 스트림의 길이가 길면 주어진 바이트 배열을 사용하여 반복하여 읽음

  - 바이트 배열의 값을 그때 그때 체크하여 읽은 값들을 다른 배열이나 메모리에 복사 내지 저장해야 함.

  - 더 이상 읽을 값이 없으면 -1 리턴

- int read(byte\[\] b, int off, int len) : 입력스트림에서 len 만큼 읽고 byte 배열 b의 off 위치부터 len 길이만큼 저장, len 길이 리턴 or 실제 읽은 길이 리턴

- 예제

  - 파일로 1bytes 씩 읽고 쓰기

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    FileInputStream fis = null;
    FileOutputStream fos = null;
    try{
      fis = new FileInputStream("test.txt");
      fos = new FileOutputStream("output.txt");
      
      int readData = -1;
      while((readData = fis.read()) != -1){
        fos.write(readData);
      }
    }catch(Exception e){
      e.printStackTrace();
    }finally{
      try{
        fos.close();
      }catch(IOException ioe){
        ioe.printStackTrace();
      }
      try{
        fis.close();
      }catch(IOException ioe){
        ioe.printStackTrace();
      }
    }
    ```

    </div>

    </div>

  - 512byte씩 읽기

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    byte[] buffer = new byte[512];
    while((readData = fis.read(buffer)) != -1){  // buffer가 -1이면 더 읽을 스트림이 없음.
      fos.write(buffer,0,readData);
    }
    ```

## OutputStream

### 종류

- FileOutputStream

- PrintStream

- DataOutputStream

- BufferedOutputStream

- ByteArrayOutputStream : 내부에 저장공간이 있어 출력시 모든 내용을 저장함. toByteArray() 바이트 배열 반환

### 공통 method

- void write(int b) : 1byte씩 출력 스트림에 전달. int형의 끝 1byte에 값을 저장하여 전달(int - 4byte 데이터형)

- void write(byte\[\] b) : byte 배열 b 출력

- void write(byte\[\] b, int off, int len) : byte 배열 b의 off 위치부터 len 길이만큼 출력

- void flush() : 출력스트림 내부 버퍼에 잔류하는 모든 바이트 출력(기본 구조 : 버퍼에 꽉 차면 출력)

- void close() : 출력 스트림 닫음

- 예제

  - write() : 바이트 길이만큼 반복해서 출력

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    OutputStream os = new FileOutputStream("sample.txt");
    byte[] data = "ABC".getBytes();
    for(int i = 0; i , data.length; i++){
      os.write(data[i]);
    }
    os.flush()
    ```

    </div>

    </div>

  - write(byte\[\] b)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    byte[] data = "ABC".getBytes();
    os.write(data);
    os.flush();
    os.close();
    ```

    </div>

    </div>

  - write(byte\[\] b, int off, int len)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    byte[] data = "ABC".getBytes();
    os.write(data,1,2); //BC 출력
    ```

# 다양한 타입 입출력

## try-with-resource 블럭 선언

- I/O 객체는 사용후 close 메소드를 호출해야 함.

- close 메소드를 호출하지 않아도 자동으로 close 하는 방법

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  try(
    DataOutputStream dos = new DataOutputStream(new FileOutputStream("output.txt"));
  ){
    dos.writeInt(100);
  }catch(Exception e){

  }
  ```

  </div>

  </div>

## 다양한 타입으로 저장할 수 있는 DataOutputStream

- 정수 저장 : writeInt()

- boolean 저장 : writerBoolean()

- double 저장 : writeDouble()

- byte 저장 : writerByte()

- float 저장 : writeFloat()

## 다양한 타입 입력

- DataInputStream

  - 정수 : readInt()

  - boolean : readBoolean()

  - double : readDouble()

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  try(
    DataInputStream dis = new DataInputStream(new FileInputStream("output.txt'));
  ){
    int i = dis.readInt();  //파일 내용이 이 순서대로 저장되어 있어야 함
    boolean b = dis.readBoolean();
    double d = dis.readDouble();
  }catch(Exception e){
    e.printStackTrace();
  }
  ```
