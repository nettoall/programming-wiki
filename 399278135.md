# File 입출력

- 파일의 크기, 속성, 이름 등의 정보 제공 기능 및 생성,삭제 기능 제공

## 파일 입출력 처리

- FileReader, FileWriter : 텍스트 파일

- FileInputStream, FileOutputStream : 바이너리 파일

- 개행문자 : CR(Carriage Return, 0x0D), LF(Line Feed, 0x0A)

- 유닉스/리눅스, 맥에서 개행은 LF만 사용

- 예제(text File)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  BufferedReader br = null;
  PrintWriter pw = null;
  try(
    br = new BufferedReader(new FileReader("src/exam01/file.txt'));
    pw = new PrintWriter(new FileWriter("test.txt"));
  ){
    String line = null;
    while((line = br.readLine()) != null){
      pw.println(line);
    }
  }catch(Exception e){
    e.printStackTrace();
  }
  ```

  </div>

  </div>

- 예제(binary File)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  InputStream is = new FileInputStream(inputFile);
  OutputStream out = new FileOutputStream(outputFile);
  byte[] buffer = new byte[buffer_size];

  while((readLen = is.read(buffer)) != -1){
    out.write(buffer,0, readLen);
  }
  is.close();
  out.close();
  ```
