# Web 관련 모듈

## HTTP Module

- HTTP 프로토콜을 사용하여 data를 송수신할 수 있는 모듈

- 사용법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');
  ```

  </div>

  </div>

### Node.js 를 Web Server 로 사용

- http 모듈의 createServer() 함수로 HTTP Server 생성

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');

  http.createServer(funtion(req, res) {
    res.write('Hello World');
    res.end();
  }).listen(8080);
  ```

  </div>

  </div>

  - res.write() : response 에 메시지 정보 출력하기

  - listen() : 서버 listen 하도록 호출

### HTTP Header 추가하기

- response에 writeHead() 메소드로 header 추가

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');

  http.createServer(funtion(req, res) {
    res.writeHead(200, {'Content-Type':'text/html'});
    res.write('Hello World');
    res.end();
  }).listen(8080);
  ```

  </div>

  </div>

  - res.writeHead() 메소드로 header 정보 추가

  - http 상태코드 200은 정상일 때, 응답 헤더 정보 content type 정의

### Query String 읽기

- url로부터 query string을 읽을 때 req 파라미터로부터 읽어 들임

- req.url 속성으로부터 정보를 읽음

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');

  http.createServer(funtion(req, res) {
    res.writeHead(200, {'Content-Type':'text/html'});
    res.write(req.url); // req에서 url 정보를 출력함
    res.end();
  }).listen(8080);
  ```

  </div>

  </div>

## URL Module

- Query String 으로부터 **필요한 요소들을 분리하기 위해**서는 URL Module을 사용

- url.parse()는 Node v16부터 deprecated 됨(**아래 예제들 URL 모듈 사용하는 방식으로 변경 필요**)

- URL 모듈에는 host, pathname, search, query 등을 지원함

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var url = require('url');
  var adr = 'http://localhost:8080/default.htm?year=2022&month=November';
  var q = url.parse(adr, true);

  console.log(q.host); // localhost:8080
  console.log(q.pathname); // /default_htm
  console.log(q.search); // ?year=2022&month=November

  var qdata = q.query;
  console.log(qdata.month); // November
  ```

  </div>

  </div>

- http 서버에서 url로 query String 출력 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');
  var url = require('url');
  // http://localhost:8080/?year=2022&month=November
  http.createServer(function(req, res) {
    res.writeHead(200, {'Content-Type':'text/html'});
    var q = url.parse(req.url, true).query;
    var txt = q.year + " " + q.month;
    res.end(txt);
  }).listen(8080);
  ```

  </div>

  </div>

- url.parse() deprecated된 예제에서 URL 모듈 사용한 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');
  var url = require('url');
  // 예제 URL : http://localhost:3000/?year=2022&month=November
  http.createServer(function (req, res) {
      var _url = req.url;
      var queryData = new URL('http://localhost:3000' + _url).searchParams;
      var txt = queryData.get('year') + " " + queryData.get('month');
      res.writeHead(200, { 'Content-Type': 'text/html' });
      res.end(txt);
  }).listen(3000);
  ```

  </div>

  </div>

## 동적인 웹페이지 만들기

- 페이지에서 클릭을 하면 url의 id 값을 받아 response.end에 출력되는 html의 title 값을 변경

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var http = require('http');
  var fs = require('fs');
  var url = require('url');
   
  var app = http.createServer(function(request,response){
      var _url = request.url;
      var queryData = url.parse(_url, true).query;
      var title = queryData.id;

        title = 'Welcome';
      }
      if(_url == '/favicon.ico'){
        return response.writeHead(404);
      }
      response.writeHead(200);
      var template = `
      <!doctype html>
      <html>
      <head>
        <title>WEB1 - ${title}</title>
        <meta charset="utf-8">
      </head>
      <body>
        <h1><a href="/">WEB</a></h1>
        <ul>
          <li><a href="/?id=HTML">HTML</a></li>
          <li><a href="/?id=CSS">CSS</a></li>
          <li><a href="/?id=JavaScript">JavaScript</a></li>
        </ul>
        <h2>${title}</h2>
        <p><a href="https://www.w3.org/TR/html5/" target="_blank" title="html5 speicification">Hypertext Markup Language (HTML)</a> is the standard markup language for <strong>creating <u>web</u> pages</strong> and web applications.Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages.
  ```
