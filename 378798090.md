## 파이썬에서 정규표현식 사용

- import re

- re.compile(“정규표현식”)

## 정규 표현식의 기초, 메타문자

- 메타문자 : .^\$\*+?{}\[\]\\()

### 문자 클래스

- \[\] : \[\] 사이의 문자들과 매치

- \[-\] : from - to

- ^: not

- 예제 : \[a-zA-Z\] 알파벳 모두, \[0-9\] 숫자 모두

- 자주 사용하는 문자클래스

<div class="table-wrap">

|                 |                                                           |
|-----------------|-----------------------------------------------------------|
| **정규 표현식** | **설명**                                                  |
| \d              | 숫자와 매치, \[0-9\]와 동일                               |
| \D              | 숫자가 아닌 것과 매치. \[^0-9\] 와 동일                   |
| \s              | whitespace 문자(space, tab)과 매치. \[\t\n\r\f\v\]와 동일 |
| \S              | whitespace문자가 아닌 것과 매치. \[^\t\n\r\f\v\]와 동일   |
| \w              | 문자+숫자와 매치. \[a-zA-Z0-9\_\]와 동일한 표현           |
| \W              | 문자+숫자가 아닌 문자와 매치. \[^a-zA-Z0-9\]와 동일       |

</div>

### dot(.)

- \n을 제외한 모든 문자와 매치

- a.b : a+모든문자+b

- a\[.\]b : a와 b 사이에 dot(.) 문자가 있으면 매치

### 반복(\*)

- 0번 이상 반복

- ca\*t : ct, cat, caat 등 a가 0 번 이상 반복

### 반복(+)

- 1번 이상 반복

- ca+t : cat, caat 등 a가 1번이상 반복

### 반복({m,n},?)

- 반복횟수를 고정. 반복회수 m부터 n 횟수까지 매치

- {m,} 반복횟수 m번 이상

- {,n} 반복횟수 n번 이하

- 예제

  - {m} : m번 반복, ca{2}t : caat

  - {m,n} : m번부터 n번까지 반복, ca{2,5}t : caat, caaaaat

  - ? : {0,1} 와 동일, ab?c : ab, abc

### 정규표현식 지원 re 모듈

- 파이썬 정규표현식 지원하기 위해 re 모듈 제공

- 사용법

  ``` syntaxhighlighter-pre
  >>> p = re.compile('ab*')
  ```

### 정규식을 사용한 문자열 검색

- 제공하는 메소드

  - match() : 문자열의 처음부터 정규식과 매치하는지 조사

  - search() : 문자열 전체를 검색하여 정규식과 매치하는지 조사

  - findall() : 정규식과 매치되는 모든 문자열을 리스트로 돌려줌

  - finditer() : 정규식과 매치되는 모든 문자열을 반복 가능한 객체로 돌려줌

#### match

- 문자열의 처음부터 정규식과 매치되는지 조사

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import re
  >>> p = re.compile('[a-z]+') # 정규 표현식
  >>> m = p.match("python")    # 조사할 문자열
  >>> print(m)
  <re.Match object; span=(0, 6), match='python'>
  >>> m = p.match("3 python")
  >>> print(m)
  None
  ```

  </div>

  </div>

#### search

- 문자열 전체를 검색하여 매치하는지 조사

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> p = re.compile('[a-z]+')
  >>> m = p.search("python")
  >>> print(m)
  <re.Match object; span=(0, 6), match='python'>
  >>> m = p.search("3 python")
  >>> print(m)
  ```

#### findall

### match 객체의 메서드

- group() : 매치된 문자열을 돌려준다.

- start() : 매치된 문자열의 시작 위치를 돌려준다.

- end() : 매치된 문자열의 끝 위치를 돌려준다.

- span() : 매치된 문자열의 (시작, 끝)에 해당하는 튜플을 돌려준다.

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> import re
  >>> p = re.compile('[a-z]+')
  >>> m = p.match("python")
  >>> m.group()
  'python'
  >>> m.start()
  0
  >>> m.end()
  6
  >>> m.span()
  (0, 6)
  ```

### 컴파일 옵션

- 정규식 컴파일 옵션

- DOTALL : 약어 S, dot 문자(.)가 줄바꿈 문자를 포함하여 모든 문자와 매치

- IGNORECASE: 약어 I, 대.소문자에 관계없이 매치

- MULTILINE: 약어 M, 여러 줄과 매치

- VERBOSE: 약어 X, verbose 모드를 사용한다.
