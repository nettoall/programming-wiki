# Thread 생성과 실행

## Thread 클래스로부터 직접 생성(Runnable interface)

- Thread를 사용하기 위해서는 Thread에 의해 실행될 코드를 작성 - task

- Runnable 인터페이스를 구현하여 인자 없는 run() 메소드를 구현(첫 번째 방법)

- Runnable 구현 클래스 작성

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Task implemtns Runnable {
    public void run() {
      // Thread 실행 코드 
    }
  }

  class a {
    public static voia main(String[] args) {
      // main
      Runnable task = new Task();

      Thread thread = new Thread(task); // Runnable task를 매개값으로 Thread 생성자 호출
      thread.start();
    }
  }
  ```

  </div>

  </div>

- 익명 구현 객체로 구현

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Thread thread = new Thread(new Runnable(){
    public void run() {
      // Thread 실행 코드
    }
  });
  thread.start();
  ```

  </div>

  </div>

- Runnable interface는 run() 메소드 하나만 정의됨. 람다식으로 변환 가능

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre

    // Thread 실행 코드
  });
  thread.start();
  ```

## Thread 하위 클래스로부터 생성

- Thread 클래스를 상속한 후 run 메소드를 재정의

- Thread 클래스로 작업하는 것은 주의하지 않으면 지루하고도 오류가 발생하기 쉽다.

- Thread 클래스를 상속받아 Thread 구현(두 번째 방법)

# 스레드 우선순위

- 멀티 스레드는 동시성(Concurrency) 또는 병렬성(Parallelism) 으로 실행됨

  - 동시성 : 멀티 작업을 위해 하나의 코어에서 멀티 스레드가 번갈아 가면 실행

  - 병렬성 : 멀티 작업을 위해 멀티 코어에서 개별 스레드를 동시에 실행

- 스레드의 개수가 코어의 수보다 많을 경우, 스레드를 어떤 순서에 의해 동시성으로 실행할 것인가를 결정 : 스레드 스케줄링

- 자바의 스레드 스케줄링은 우선순위(Priority) 방식과 순환 할당(Round-Robin) 방식 사용

  - 우선순위 방식 : 우선 순위가 높은 스레드가 실행 상태를 더 많이 가지도록 스케줄링 → 코드 제어 가능

  - 순환 할당 : 시간 할당량을 정해서 스레드마다 정해진 시간만큼 교대로 수행 → jvm에 의해 결정

- 자바의 우선순위 설정

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  thread.setPriority(우선순위);

  thread.setPriority(Thread.MAX_PRIORITY); // 10
  thread.setPriority(Thread.NORM_PRIORITY); // 5 
  thread.setPriority(Thread.MIN_PRIORITY);  // 1
  ```

  </div>

  </div>

- 우선 순위 샘플 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  public class CalcThread extends Thread {
    public CalcThread(String name) {
      setName(name);
    }
    
    public void run() {
      for(int i = 0; i < 2000000; i++){
      }
      System.out.println(getName());
    }
  }

  public class PriorityExample {
    public static void main(String[] args) {
    
      for(int i = 1; i <= 10; i++) {
        Thread thread = new CalcThread("thread" + i);
        if(i != 10) {
          thread.setPriority(Thread.MIN_PRIORITY);
        } else {
          thread.setPriority(Thread.MAX_PRIORITY);
        }
        thread.start();
      }
    }
  }
  ```
