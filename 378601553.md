## 예외처리

- 파이썬에서 오류 발생시 처리 방법

- 사용방법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  try:
    ...
  except [발생오류[as 오류메시지변수]]:
    ...
  ```

  </div>

  </div>

- except 3가지 사용 방법

  1.  try, except 만 사용

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      try:
        ...
      except:
        ...
      ```

      </div>

      </div>

  2.  특정 오류만 처리

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      try:
        ...
      except 발생 오류:
        ...
      ```

      </div>

      </div>

  3.  특정 오류에 대한 메시지 변수 처리

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      try:
        ...
      except 발생오류 as 오류메시지변수:
        ...
        
      try:
        4/0
      except ZeroDivisionError as e:
        print(e)
      ```

      </div>

      </div>

- 사용한 리소스 close 시 finally 사용

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  f = open('foo.txt','w')
  try:
    수행
  finally:
    f.close()
  ```

  </div>

  </div>

- 여러 오류 처리시 방법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  try:
    ...
  except 발생오류1:
    ...
  except 발생오류2:
    ...
  ```

  </div>

  </div>

  - 처리방법 1

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    try:
      a = [1,2]
      print(a[3])
      4/0
    except ZeroDivisionError:
      print("0으로 나눌수 없음")
    except IndexError:
      print("인덱싱할 수 없음")  # 인덱싱 오류가 먼저 발생 zero 오류 발생 안함
    ```

    </div>

    </div>

  - 처리방법 2

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    try:
      a = [1,2]
      print(a[3])
      4/0
    except ZeroDivisionError as e:
      print(e)
    except IndexError as e:
      print(e)  
    ```

    </div>

    </div>

  - 처리방법 3

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    try:
      a = [1,2]
      print(a[3])
      4/0
    except (ZeroDivisionError, IndexError) as e:
      print(e)
    ```

    </div>

    </div>

### 오류 회피하기

- 오류가 발생시 통과시켜야 할 때 처리 방법 (pass)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  try:
    f = open("없는파일","r")
  except FileNotFoundError:
    pass // 오류 패스 예약어
  ```

  </div>

  </div>

### 오류 일부러 발생 시키기

- raise 명령어로 오류 발생

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Bird:
    def fly(self):
      raise NotImplementedError  # 오류 발생 -> Bird를 상속받은 클래스는 무조건 fly 함수를 override 하도록 일부러 예외 발생시킴
  ```

  </div>

  </div>

- 테스트 예제(Eagle.py)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Eagle(Bird)
    pass
  ```

  </div>

  </div>

  - 실행

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> eagle = Eagle()
    >>> eagle.fly() # 오류 발생
    ```

    </div>

    </div>

  - eagle fly() 함수 override 정의 후 실행시 정상 출력

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    class Eagle(Bird)
      def fly(self):
        print("very fase")
    ```

    </div>

    </div>

### 예외 만들기

- 프로그램 수행중 특별한 경우에만 예외 처리를 하기 위해 예외를 만들어서 사용

- 사용법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class MyError(Exception):
    pass
  ```

  </div>

  </div>

- 예제

  - 테스트 프로그램

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    def say_nick(nick)
      if nick == '바보':
        raise MyError()
      print(nick)
    ```

    </div>

    </div>

  - 실행

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    >>> say_nick("천사")
    천사
    >>> say_nick("바보")
    # 오류발생
    ```

    </div>

    </div>

  - MyError 발생 예외 처리

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    try:
      say_nick("천사")
      say_nick("바보")
    except MyError:
      print("허용되지 않는 별명입니다.")
    ```

    </div>

    </div>

  - 오류메시지사용 예외처리

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    try:
      say_nick("천사")
      say_nick("바보")
    except MyError as e:
      print(e)
    ```

    </div>

    </div>

    - print(e) 출력 안 됨 → 오류메시지 출력하려면 ‘\_\_str\_\_’ 메서드 구현

    - class MyError 수정 ('\_\_str\_\_' 메소드 필수 정의)

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      class MyError(Exception):
        def __str__(self):  # 오류메시지를 print 문으로 출력할 경우에 호출되는 메서드
          return "허용되지 않는 별명입니다."
      ```

      </div>

      </div>
