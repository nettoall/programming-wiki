# 파일 읽기 동기/비동기

파일 읽기 동기 처리

- fs.readFilesync() 사용

- 파일을 읽고 결과를 돌려줄 때까지 대기

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
   console.log('A');
   var result = fs.readFileSync('syntax/sample.txt', 'utf-8');
   console.log(result);
   console.log('C');
  ```

  </div>

  </div>

## 파일 읽기 비동기 처리

- fs.readFile() 사용, funtion(err, result)

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  console.log('A');
  fs.readFile('syntax/sample.txt', 'utf-8', function (err, result) {
      console.log(result);
  });
  console.log('C');
  ```

  </div>

  </div>

- File을 읽고 결과를 가져올 때까지 대기하는 것이 아닌 다른 업무 실행

## 콜백

- fs.readFile() 함수에서 function() 으로 정의된 부분을 콜백함수라 함

- 작업을 처리하고 나서 특정 함수를 호출하도록 처리

- 간단한 콜백 구현 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var a = function () {
      console.log('A');
  }

  function slowfunc(callback) {
      callback();
  }

  slowfunc(a);
  ```

  </div>
