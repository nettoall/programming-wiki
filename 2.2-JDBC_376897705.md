### SQL 파라미터

- SimpleJdbcTemplate에 작업 요청시 문자열로 된 SQL 제공

- binding 값이 있는 경우 '?' 치환자 입력. 파라미터 바인딩 사용

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  # ? 치환자
  INSERT INTO MEMBER(ID, NAME, POINT) VALUES(?, ?, ?);

  # 이름 치환자
  INSERT INTO MEMBER(ID, NAME, POINT) VALUES(:id, :name, :point);
  ```

  </div>

  </div>

- Map 이나 Object에 담긴 내용을 키 값이나 프로퍼티 이름을 이용해 바인딩(프로퍼티 소스 사용)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  // Map/MapSqlParameterSource
  Map<String, Object> map = new HashMap<String, Object>();
  map.put("id", 1);
  map.put("name","Spring");
  map.put("point", 3.5);
  SimpleJdbcTemplate.update("INSERT INTO MEMBER(ID, NAME, POINT) VALUES(:id, :name, :point)", map);
  ```

  </div>

  </div>

- 코드를 이용해 맵에 정보를 직접 입력(???)

- SpringMapSqlParameterSource 사용 법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  MapSqlParameterSource params = new MapSqlParameterSource()
                                 .addValue("id", 1)
                                 .addValue("name", "spring")
                                 .addValue("point", 3.5);
  ```

  </div>

  </div>

### SQL 조회

- 코드 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  int count = SimpleJdbcTemplate.queryForInt("select count(*) from member");
  SimpleJdbcTemplate.queryForInt("select count(*) from member where point > : min", new MapSqlParameterSource("min",min));
  queryForLong(sql, parameter)
  <T> queryForObject(String sql, Class<T> requiredType, parameter)
  String name = SimpleJdbcTemplate.queryForObject("select name from memeber where id=?", String.class, id);
  <T> T queryForObject(sql, RowMapper<T> rm, parameter)
  ```

- RowMapper interface 를 구현해서 만든 콜백 Object

- 테이블의 컬럼 이름과 매핑될 Object의 프로퍼티 이름이 일치할 경우 BeanPropertyRowMapper\<T\> 사용 권장

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Member m = SimpleJdbcTemplate.queryForObject("select * from member where id= ?", new BeanPropertyRowMapper<Member>(Member.class), id);
  ```

  </div>

  </div>
