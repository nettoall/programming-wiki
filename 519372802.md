# 환경설정

- pyenv 설치: python의 다양한 버전을 바꿔가면서 사용할 수 있게 해줌

## pyenv 설치

- powehsell 관리자 모드로 열시

- 명령어 입력

  - Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine

  - Invoke-WebRequest -UserBasicParsing -Uri “<span rel="nofollow">https://raw.githubusercontent.com/pyenv-win/pyenv-win/master/pyenv-win/install-pyenv-win.ps1</span>” -OutFile “./install-pyenv-win.ps1"; &”./install-pyenv-win.ps1"

- 새로운 powershell 열기

  - pyenv install -l : 설치 가능한 버전 조회

  - pyenv install 3.12.4

  - pyenv global 3.12.4 : 전역적으로 사용할 python 버전 지정

  - python --version

  - pyenv which python : 실제 사용하는 python 확인

- 특정 디렉토리에 다른 버전 사용하기

  - pyenv install 3.9.13

  - mkdir python_projects && cd python_projects

  - pyenv local 3.9.13

  - python --version

- pyenv 명령어

  - pyenv versions : 현재 설치된 python 버전들

  - pyenv vname: 현재 사용중인 python 버전

  - pyenv update : pyenv의 python 설치 항목 최신화

  - &”\${env:PYENV_HOME}\install-pyenv-win.ps1” : pyenv 업그레이드

- 텍스트 에디어 글짜 크기 조정

  - Editor: Mouse Wheel Zoom 체크
