# <span id="title-text"> Programming : 네트워크 </span>

</div>

<div id="content" class="view">

<div class="page-metadata">

Created by <span class="author"> Dongwook Han</span>, last modified on 7월 31, 2023

</div>

<div id="main-content" class="wiki-content group">

getClientIp를 할 때 보안 문제로 방화벽 등 사용으로 인해 요청이나 응답이 가공되오 나가고 들어오기 때문에 X-Forwarded-For 헤더 정보로 client ip를 가져오는 경우 있음 : 클라리언트의 원 ip 주소를 식별하는 표준 헤더 또는 getRemoteAddr() 을 사용

가끔 Ip가 0:0:0:0:0:0:0:1 와 같이 찍히는데 IPv6 주소임

로컬환경 같은 경우, 서버를 시작할 때 -Djava.net.preferIPv4Stack=true 를 추가해주면 클라이언트 IP 가져올 때 IPv4로 가져옴
