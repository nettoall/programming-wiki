# 이벤트

- Node.js는 event-driven application 임

- 접속이 일어나거나 파일을 열거나 하는 것이 이벤트임

- Node.js에서는 Events 라는 모듈을 제공함

- create, fire, listen 등 이벤트에 대응하는 모듈임

- 모든 이벤트 속성과 메소드는 EventEmitter 오브젝트의 인스턴스이며, 사용하기 위해서 EventEmitter 를 생성해야 함.

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var events = require('events');
  var eventEmitter = new events.EventEmitter();
  ```

  </div>

  </div>

## EventEmitter Object

- EventEmitter 를 가지고 event에 Event Handler를 결합할 수 있다.

- scream event가 발생할 때 실행되는 메소드를 생성하는 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  var events = require('events');
  var eventEmitter = new events.EventEmitter(); 

  // Create an event handler
  var myEventHandler = function() {
    console.log('I hear a scream!');
  }

  // Assign the event handler to an event;
  eventEmitter.on('scream', myEventHandler);

  // Fire the 'scream' event
  eventEmitter.emit('scream');
  ```

  </div>
