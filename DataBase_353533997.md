# SQL

## Join

### 정의

- 동일 조건에서 조인 조건을 만족하는 값이 없는 행들을 조회하기 위해 outer join 사용

- 조인시 값이 없는 조인 측에 <img src="./media/images/icons/emoticons/add.png" class="emoticon emoticon-plus" data-emoji-id="atlassian-plus" data-emoji-shortname=":plus:" data-emoji-fallback=":plus:" data-emoticon-name="plus" width="16" height="16" alt="(더하기)" /> 위치

### LEFT OUTER JOIN

- 오른쪽 테이블에 조인시킬 컬럼의 값이 없는 경우(empty)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  SELECT DISTINCT(e.deptno), d.deptno
  FROM dept d
  LEFT OUTER JOINT emp e
  ON   d.deptno = e.deptno;
  ```

  </div>

  </div>

### RIGHT OUTER JOIN

- 왼쪽 테이블(emp)에 조인시킬 컬럼의 값이 없는 경우\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  SELECT DISTINCE(e.deptno), d.deptno
  FROM   emp e
  RIGHT OUTER JOIN Dept d
  ON    e.deptno = d.deptno;
  ```

  </div>

  </div>
