## CSS선택자

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
<html>
<head>
<style>
a {
    color:black;
    text-decoration: none;
    }
    .saw {
     color:gray;
    }
    //.active {
    //  color:red;
    //}
    #active {
      color:red;
    }
  h1 {
    font-size:45px;
    text-align:center;  
  }
</style>
</head>
<body>
 <h1<a href="index.html">WEB</a></h1>
 <ol>
  <li><a href="1.html" class="saw">HTML</a></li>
  <li><a href="2.html" class="saw active" id="active">CSS</a></li>
  <li><a href="3.html">JavaScript</a></li>
 </ol>
</body>
</html>
```

</div>

</div>

- 선택자를 지정하여 css 적용하기 위해 “.saw” class를 추가하여 지정

- class=”saw active” 두 개를 정의시 style 선언 순서에 따라 적용이 됨.

- 우선 순위가 높은 선택자를 사용하기 위해 id 선언. saw선언보다 뒤에 있어도 적용됨(id 선택자가 class선택자보다 우선함) id \> class \> tag

Block

- style 로 지정하는 각 항목별로 자기만의 block 크기를 가진다.

- h1 tag는 해당 문구의 화면을 커버하나 a tag는 해당 문자 만을 커버한다.

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
<style>
/* 각 태그별 정의 */
h1 {
   border-width:5px;
   border-style:solid;
   border-color:red;
}
a{
   border-width:5px;
   border-style:solid;
   border-color:red;
}
/* 중복항목 제거 및 재정의 */
h1, a {
  border:5px solid red;
}
</style>
```

</div>

</div>
