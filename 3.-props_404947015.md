## props

- React 의 속성 정의

- \<Header title=”REACT”\>\</Header\> 정의 시 props 를 출력하면 props를 Object로 인식

- props의 값을 가져오려면 {props.title} 로 처리 (표현식)

- 예제 (\<Header title=”…” =\> props.title 태그내 속성 정의가 props.속성)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  import logo from './log.svg';
  import './App.css';

  function Header(props) {
    console.log('props', props, props.title);
    return <header>
    <h1><a href="/">{props.title}</a></h1>
    </header>
  }
  ...
  function App() {
    return (
      <div>
        <Header title="WEB"></Header>  // React function 명은 대문자로 시작
        <Nav></Nav>
  ```

- \<Nav\> 에 속성 배열로 추가하기 (App 함수에서 정의한 배열을 param 으로 전달)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  import logo from './log.svg';
  import './App.css';

  ...
  function Nav(props) {
    // 출력할 값 정의 
    const list = []
    for(let i = 0; i < props.topics.length; i++) {
      let t = props.topics[i];
      list.push(<li key={t.id}><a href={'/read/'+ t.id}>{t.title}</a></li>)
    }
    return <nav>
    <ol>
      {list}
    </ol>
    </nav>
  }

  function App() {
    // 배열 정의
    const topics = [
      {id:1, title:'html', body: 'html is ...'},
      {id:2, title:'css', body: 'cssis ...'},
      {id:3, title:'javascript', body: 'javascript is ...'}
    ]
    return (
      <div>
        <Header title="WEB"></Header>  // React function 명은 대문자로 시작
        <Nav topics={topics}></Nav>
        <Article title="Welcome" body="Hello, WEB"></Article>
      </div>
    );
  }

  export default App;
  ```

  </div>
