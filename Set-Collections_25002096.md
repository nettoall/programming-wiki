## Set 반복자 사용 예제

Set\<String\> set = …;

Iterator\<String\> iterator = set.iterator();

- 메소드\

<div class="table-wrap">

|          |            |                               |
|----------|------------|-------------------------------|
| **반환** | **메소드** | **설명**                      |
| booelan  | hashNext() | 가져올 객체 여부              |
| E        | next()     | 컬렉션에서 하나의 객체 가져온 |
| void     | remove()   | Set컬렉션에서 객체 제거       |

</div>
