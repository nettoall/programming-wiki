## Class

- class 는 java 등의 클래스와 동일한 의미

- class의 필요성

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  result = 0

  def add(num):
    global result  // 결과값을 유지하기 위해 전역변수 선언
    result += num
    return result
  ```

  </div>

  </div>

  - 위와 같이 함수로 동시에 여러 개를 계산하려면 일반적으로 add(), add2() 함수를 선언해서 처리해야 함.

- class 정의를 하면 두고두고 사용 가능

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Calculator:
    def __init__(self):
      self.result = 0  // self는 자료형이 무엇인가?(개인질문) -> 자기 객체가 자동 전달됨
      
    def add(self, num)
      self.result += num
      return self.result
      
  cal1 = Calculator()
  cal2 = Calculator()
  print(cal1.add(3))
  print(cal2.add(4))
  ```

  </div>

  </div>

- type(객체) : 어떤 클래스 타입인지 알려주는 함수

- 클래스, 객체, 인스턴스란 정의도 python에서 사용함

### 클래스내 메소드 정의

- 메소드 정의

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class FourCal:
    def setData(self, first, second): // self에는 객체 a가 자동으로 전달됨
      self.first = first
      self.second = second
      
  a = FourCal()
  a.setData(4,2)  // self 값 지정 안 해도 a가 지정됨
  ```

  </div>

  </div>

- 클래스내 메소드의 첫번째 인자는 항상 self 정의(다른 키워드 가능하나 일반적으로 self 사용)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  FourCal.setData(a,4,2) // 로 정의 가능
  a = Fourcla()
  a.setData(4,2)
  print(a.first)  // 변수를 정의 안 해도 메소드에 정의하면 인스턴스의 변수명으로 출력가능
  ```

  </div>

  </div>

- id(a.first) : a의 fisrt 주소값을 확인

- 예제(사칙연산)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Calculator:
    def setData(self, first, second): // 메소드 첫번째 인자는 항상 객체 지정
      self.first = first
      self.second = second
      
    def add(self):
      result = self.first + self.second
      return result
      
    def mul(self):
      result = self.first * self.second
      return result
      
    def sub(self):
      result = self.first - self.second
      return result
      
    def div(self):
      result = self.first / self.second
      return result
  ```

  </div>

  </div>

- self 는 java의 this 와 같은 의미로 판단해도 될 듯

### 생성자

- 앞에서 정의한 클래스에서 setData()로 값 설정 안 하고 add() 호출시 에러가 발생함

- class 가 호출되었을 때 기본값이 설정되도록 생성자 선언

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Calculator:
    def __init__(self, first, second):
      self.first = first
      self.second = second
  ```

  </div>

  </div>

  - 사칙연산 예제에서 setData() 를 ‘**\_\_init\_\_’ 로 재정의**

  - 객체 생성시 자동으로 생성자 호출

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    a = Calculator(4,2) // 인자 없이 정의시 오류 발생
    ```

    </div>

    </div>

### 클래스 상속

- 클래스를 상속받아 새로운 클래스를 정의할 수 있다.

- 사용법 : class 클래스 이름(**상속할 클래스이름**)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class MyCal(Calculator):
    def pow(self):
      result = self.first ** self.second // first의 second 승
      return result
  ```

  </div>

  </div>

### 클래스 변수

- 객체변수 : 각 객체마다 고유한 값

- 사용 예

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  class Family:
    lastname = "김"  // 변수만 선언하지 않고 값 입력이 필수? -> java의 상수 선언과 동일
  ```

  </div>

  </div>

- 클래스 변수는 모든 객체에 공유됨

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> a = Family()
  >>> b = Family()
  >>> print(a.lastname)
  김
  >>> print(b.lastname)
  김
  >>> Family.lastname = "박"
  >>> print(a.lastname)
  박
  >>> print(b.lastname)
  박
  ```
