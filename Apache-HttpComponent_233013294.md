- createDefault(), createMinimal() 은 모두 PoolingHttpClientConnectionManager() 를 사용

- createDefault() 계속 호출시 PoolingHttpClientConnectionManger 를 계속 생성함.

- internalHttpClient와 PoolingHttpClientConnectionManger는 @ThreadSafe 함

- spring 예제(한번 생성하고 다른 클래스에서 공유해서 사용 가능)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  @Bean
  public HttpClient httpClient() {
    // HttpClient 생성 
    return HttpClient.createDefault();
  }

  -------------------------
  // 다른 class
  @Autowired
  private HttpClient httpClient;

  public void call() {
    HttpResponse response = null;
    try {
      response = httpClient.execute(new HttpGet("http://www.test.com"));
      String responseText = EntityUtils.toString(response.getEntity());
    } catch(Exception e) {
    
    } finall {
      HttpClientUtils.closeQuietly(response);
    }
  }
  ```

  </div>

  </div>

- 공유해서 사용하면 PoolingHttpClientConenctionManger 로 관리되는 instance를 같이 사용

- 필요시 다른 instance를 생성하여 사용 가능

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  @Bean
  public HttpClient httpClient {
    return HttpClient.createDefault();
  }

  @Bean
  public HttpClient PerformClient {
    PoolingHttpClientConenctionManager cm = new PoolingHttpClientConnectionManager()
        .setMaxTotal(300)
        .setDefaultmaxPerRoute(50);
        
    return HttpClients.custom().setConnectionManager(cm).build();
  }

  @Bean
  public HttpClient TimeClient {
    SoccketConfig sc = SocketConfig.custom()
    .setSoTimeout(2000)
    .setSoKeepAlive(true)
    .setTcpNodelay(true)
    .setSoReuseAddress(true)
    .build();
    return HttpClients.custom().setDefaultSocketConfig(sc).build();
  }
  ```
