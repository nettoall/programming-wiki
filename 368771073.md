## 함수

### 함수 구조 정의

<div class="code panel pdl" style="border-width: 1px;">

``` syntaxhighlighter-pre
def 함수이름(매개변수):

  수행할 문장2
```

</div>

</div>

#### 예제

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">


### 매개변수와 인수

- parameter : 함수에 입력으로 정의된 변수

- argument : 함수에 전달되는 입력값

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  def add(a, b):    # a,b 매개변수
    return a + b

  print(add(3,4))  ## 3,4 인수 
  ```

  </div>

  </div>

### 함수의 형태

자바와는 다르게 parameter, return 값의 type을 정의 안 함

- 파이썬은 매개변수 순서에 상관없이 전달 가능(매개변수를 지정하여 값 설정 시)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  result = add(b=5, a=3)
  ```

  </div>

  </div>

- parameter가 여러 개인 경우 선언(매개변수에 \* 정의 : \*args)

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  def add_many(*args):
    result = 0;
    for i in args
      result = result + i
    return result
    
  add_many(1,2,3,4,5,6,7)
  ```

  </div>

  </div>

  <div class="codeContent panelContent pdl">


## 키워드 파라미터 (\*\*)

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
def print_kwargs(**kwargs):
  print(kwargs)
```

</div>

</div>

- \*\*를 붙이면 kwargs는 딕셔너리가 됨

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  print_kwargs(a=1)
    {'a}:1}
  print_kwargs(nam='foo',age=3)
    {'age':3,'nam':'foo'}
  ```

  </div>

  </div>

### 함수의 결괏값

함수 내 return 하는 값을 여러 개 정의 시 튜플 형태로 결괏값을 반환함(결괏값은 튜플 자료형 하나임)

- 함수 선언

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  def add_and_mul(a,b):
    return a+b, a*b   # 반환 두개 정의
  ```

  </div>

  </div>

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  >>> result = add_and_mul(3,4)
  result = (7,12) # 튜플 형식으로 리턴됨
  # 두개의 결과값으로 받으려면
  >>> result1,result2 = add_and_mul(3,4)
  result1 = 7
  result2 = 12
  ```

  </div>

  </div>

### 함수안에서 선언한 변수 범위

- 함수안 변수의 범위

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  a = 1
  def vartest(a):
    a = a + 1 # a는 함수밖의 a와 별개 변수
    
  vartest(a)
  print(a)
  ```

  </div>

  </div>

- 함수밖 변수 변경하기

  1.  return사용

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      a = 1
      def vartest(a):
         a = a + 1
         return a
         
      a = vartest(a)
      ```

      </div>

      </div>

  2.  글로벌 변수 선언

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      a = 1
      def vartest():
        global a # 함수밖의 a를 사용한다는 선언
        a = a + 1

      vartest()
      ```

      </div>

      </div>

### print()

- ““ 로 둘러 싸인 문자열은 + 연산과 동일

- 문자열 띄어쓰기는 , 로 함

- 한 줄에 결괏값 출력하기 위해서는 매개변수 end 를 사용해 끝 문자 지정

- 사용방법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  print("출력")
  print("출력","은","왕이다.") # ,띄어쓰기
  print(i, end= ' ') # 한줄로 이어쓰기
  ```

## 파일 읽기 쓰기

### 파일 생성

- 사용법

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  f = open("새파일.txt",'w') # 파일열기 모드 r,w,a
  f.close()
  ```

  </div>

  </div>

<!-- -->

- 디렉토리 지정 파일 생성

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  f = open("C:/doit/새파일.txt",'w')
  f.close() # 
  ```

  </div>

  </div>

  - f.close() : 프로그램 종료시 자동으로 파일 받음

  - 파일열기모드 w : 쓰기, r : 읽기, a : 추가

  - 쓰기 모드로 연 파일을 다른 모드로 사용할 시에는 오류가 발생하므로 사용하고 난 뒤 f.close()를 선언하여 닫아 주는게 좋음

- 파일에 저장하기

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  f = open("C:/doit/새파일.txt",'w')
  for i in range(1,11):
    data = "%d번째 줄입니다.\n" % i
    f.write(data)
  f.close()
  ```

  </div>

  </div>

### 파일 읽기

- 사용법 예제

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  f = open("C:\doti\읽을파일.txt",'r')
  line = f.readLine() #첫번째 줄만 읽음
  ```

- 여러 줄 읽기

  1.  readLine()

      <div class="code panel pdl" style="border-width: 1px;">
