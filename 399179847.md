# 문자(Char) 입출력

## Reader

- 문자기반 입력

### 종류

- FileReader

- BufferedReader

- InputStreamReader

### 공통Method

- int read() : 한개의 문자(2 byte)를 읽고 리턴. int를 char로 변환하면 문자.

- int read(char\[\] buf) : char 배열에 저장하고 읽은 문자수 리턴

- int read(char\[\] buf, int off, int len) : char 배열의 off 위치부터 len길이 만큼 저장

- 예제

  - read()

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    Reader reader = new FileReader("sample.txt");
    int readData;
    while((readData = reader.read()) != -1){
      char read = (char)readData;
    }
    ```

    </div>

    </div>

  - read(char\[\] cbuf)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    int readCharNo; String data = "";
    char[] cbuf = new char[2];
    whlie((readCharNo = reader.read(cbuf)) != -1){
      data += new String(cbuf, 0, readCharNo);
    }
    reader.close()
    ```

    </div>

    </div>

  - read(char\[\] cbuf, int off, int len)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    Reader reader = new FileReader("sample.txt");
    char[] cbuf = new char[100];
    int readCharNo = reader.read(cbuf,0,100);
    ```

    </div>

    </div>

## Writer

- 문자기반 출력 스트림

### 종류

- FileWriter

- BufferedWriter

- PrintWriter

- OutputStreamWriter

### 공통Method

- void write(int c) : 출력 스트림에 문자 전달(int 뒤 2 byte)

- void write(char\[\] cbuf) : char 배열로 문자를 스트림에 전달

- void write(char\[\] cbuf, int off, int len)

- void write(String str) : 문자열을 스트림에 전달

- void write(String str, int off, int len)

- 예제

  - write()

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    Writer writer = new FileWriter("output.txt");
    char[] data = "홍길동".toCharArray();
    for(int i = 0; i , data.length; i++){
      writer.write(data[i]);
    }
    ```

    </div>

    </div>

  - write(char\[\] cbuf)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    char[] data = "홍길동".toCharArray();
    writer.write(data);
    ```

    </div>

    </div>

  - write(char\[\] cbuf, int off, int len)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    char[] data = "홍길동".toCharArray();
    writer.write(data, 1,2); // 길동
    ```

    </div>

    </div>

  - write(String str), write(String str, int off, int len)

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    String str = "안녕 홍길동";
    writer.write(str);
    writer.write(str, 3,2); //홍길
    ```
