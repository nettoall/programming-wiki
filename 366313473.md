### foreach

- 연속된 항목을 순차적으로 흝어 나가기 위해 int 타입의 변수를 사용할 필요 없음

- 이전 방식\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  float f[] = new float[10];
  for(int i = 0; i < 10; i++) {
    f[i] = rand.nextFloat();
  }

  for(float x : f)  // for int 로 배열의 값을 일일이 읽어서 처리할 필요 없이 foreach 방식으로 처리
    System.out.println(x);   // foreach 사용 예 
  ```

  </div>

  </div>

- 배열을 반환하는 메소드에서는 foreach 사용할 가능성이 큼

- String class의 toCharArray() 예\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  for(char c : "An African Swallot".toCharArray())

  ```

  </div>

  </div>

- 객체에서 iterable 인터페이스 구현한 객체도 foreach 사용 가능

### 분기

- return, break, continue

- return 사용 시점

  1.  메소드 실행 종료 후 값 반환

  2.  메소드 실행을 중단하고 그 시점의 값을 반환하는데 사용

- break, continue

  - 반복 Loop의 실행을 제어

  - break : Loop 내의 나머지 명령어를 실행하지 않고 Loop 종료

  - continue : Loop 내의 나머지 명령어를 실행하지 않고 Loop의 처음으로 가서 그 다음 반복을 계속 수행\

    <div class="code panel pdl" style="border-width: 1px;">

    <div class="codeContent panelContent pdl">

    ``` syntaxhighlighter-pre
    for(int i  = 0; i < 100; i++) {
      if(i == 74) break;   // 74일 때 Loop 종료
      if(i %9 != 0) continue;  // i % 9 일때마다 아래 System.out.println 실행 안하고 반복 수행
      System.out.println(i + " ");
      
    }
    ```

    </div>

    </div>

- while(true), for(;;) 컴파일러는 같은 방법으로 처리(의미를 알수 없음)

#### 라벨 식별자

- 자바에서 라벨은 반복문 앞에서 쓰임

- switch 문 사용시 라벨 사용\

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  label1:
    반복문 1{
      반복문 2 {
        ...
        break; // (1)
        ......
        continue; // (2)
        ......
        continue label1; // (3)
        ....
        break label1; // (4)
      }
    } (5)
  ```

  </div>

  </div>

\(1\) 번의 break 는 반복문2를 벗어나 반복문1의 Loop를 계속 실행

\(2\) 번은 반복문2를 계속 반복

\(3\) label1으로 이동 후 반복문1 반복

\(4\) label1으로 이동 후 반복문 실행안함 (5)로 이동
