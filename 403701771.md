# 경로 정의 (Path)

- java.nio.file.Path interface

- 파일의 경로를 지정하기 위해 사용

- NIO에서 **File, FileSystem을 표시**, 혹은 **파일의 경로를 표시**하는 객체

- 파일 경로, 디렉토리 구조, 파일 시스템, 파일 명 등

# Files

- java.nio.file.Files 클래스

- 파일, 디렉토리의 생성 및 삭제, 속성(읽기,숨김,크기, 소유자 등)을 읽는 메소드 제공

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  // path 정보가 없을 시 
  Files.notExist(path);
  // path 디렉토리 생성
  Files.createDirectories(path);
  // Path 파일 생성
  Files.createFile(path);
  // path 경로의 모든 파일, 디렉토리의 directory stream을 반환
  Files.newDirectoryStream(path);
  //디렉토리 여부
  Files.isDirectory(path);
  // 파일 사이즈
  Files.size(path);
  // 파일 복사
  Files.copy(path1, path2, StandardCopyOption.REPLACE_EXISTING);
  ```

  </div>

  </div>

- Stream과 Reader/Writer를 리턴하는 메소드 제공

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  Path path = Paths.get("C:\temp");
  DirectoryStream<Path> directoryStream = Files.newDirectoryStream(path);

  for(Path path: directoryStream) {
    if(Files.isDirectory(path)) {
      System.out.println("디렉토리 " + path.getFileName());
    } else {
      System.out.println("파일 " + path.getFileName() + "크기 : " + Files.size(path));
  ```

# WacthService

- 등록한 Object의 변경 및 이벤트를 감시함 → 디렉토리 내용을 수시로 표시하기 위해 해당 디렉토리를 모니터링하거나 사용하기 위해서 해당 directory를 watchService에 등록(파일 편집시 외부에서 파일이 수정되었을 때 감지하여 다시 불러올 것인지 알려주는 기능 등)

  1.  WatchService 생성

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      WatchService watchService = FileSystems.getDefault().newWatchService();
      ```

      </div>

      </div>

  2.  **모니터링할 디렉토리를 watchService에 등록**

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      Path directory = Paths.get(…);
      directory.register(watchServive, StandardWatchEventKinds.ENTRY_CREATE, // 모니터링할 유형 지정
                                       StandardWatchEventKinds.ENTRY_MODIFY,
                                       StandardWatchEventKinds.ENTRY_DELETE);
      ```

      </div>

      </div>

  3.  이벤트가 발생하면 WatchService는 **watch key를 전달(queue에 저장)**

  4.  프로그램은 무한 루프를 돌면서 queue에 watch key가 있는지 확인

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      while(true) {
        WatchKey watchKey = watchService.take(); // queue 에 watch key 들어오면 처리
      }
      ```

      </div>

      </div>

  5.  상세한 이벤트(다중 이벤트-create, modify,delete 등-) 정보는 **pollEvent()** 메소드로 가져옴

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      List<WatchEvent<?>> list = watchKey.poolEvents();
      ```

      </div>

      </div>

  6.  이벤트가 발생한 파일, 디렉토리 정보는 context()로 얻음

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      for(WatchEvent watchEvent : list) {
        //이벤트 종류 얻기
        Kind kind = watchEvent.kind();
        // 감지된 Path 얻기
        Path path = (Path)watchEvent.context();
        // 이벤트 종류별로 처리
        if(kind == StandardWatchEventKinds.ENTRY_CREATE) {
          ...
        } else if(kind == StandardWatchEventKinds.ENTRY_DELETE){
          ...
        } else if(kind == StandardWatchEventKinds.ENTRY_MODIFY) {
          ...
        } else if(kind == StandardWatchEventKinds.OVERFLOW) { // 이벤트가 소실되었거나 버려진 경우 별도의 처리코드 없음
          
        }
      }
      ```

      </div>

      </div>

  7.  reset() : 전달받은 watchKey를 리셋하고 다음 이벤트를 받을 수 있게 처리 → 파일 삭제 등의 이유로 모니터링이 불가능하면 false 를 리턴하여, 모니터링하는 로직에서 나가도록 할 수 있음

      <div class="code panel pdl" style="border-width: 1px;">

      <div class="codeContent panelContent pdl">

      ``` syntaxhighlighter-pre
      while(true) {
        WatchKey watchKey = watchService.take();
        List<WatchEvent<?>> list = watchKey.poolEvents();
        
        for(WatchEvent watchEvent : list) {
          ...
        }
        boolean valid = watchKey.reset();
        if(!valid) {break;}
      }
      watchService.close();
      ```

      </div>

      </div>

- 예제

  <div class="code panel pdl" style="border-width: 1px;">

  <div class="codeContent panelContent pdl">

  ``` syntaxhighlighter-pre
  public class WatchServiceExample extends Application {
    class WatchServiceThread extends Thread {
      @Override
      public void run() {
        try {
          WatchService watchService = FileSystems.getDefault().newWatchService();
          Path directory = Paths.get("C:/Temp");
          directory.register(watchService, StandardWatchEventKinds.ENTRY_CREATE,
                                           StandardWatchEventKinds.ENTRY_DELETE,
                                          StandardWatchEventKinds.ENTRY_MODIFY);
          
          while(true) {
            WatchKey watchKey = watchService.take(); 
            List<WatchEvent<?>> list = watchKey.pollEvents();
            
            for(WatchEvent watchEvent : list) {
              Kind kind = watchEvent.kind();
              Path path = (Path)watchEvent.context();
              
              if(kind == StandardWatchEventKinds.ENTRY_CREATE) {
                Platform.runLater(() -> textArea.appendText("파일 생성됨 -> " + 
                                                         path.getFileName() + "\n"));
              } else if(kind == StandardWatchEventKinds.ENTRY_DELETE) {
                Platform.runLater(() -> textArea.appendText("파일 삭제됨 -> " + 
                                                         path.getFileName() + "\n"));
              } else if(kind == StandardWatchEventKinds.ENTRY_MODIFY) {
                Platform.runLater(() -> textArea.appendText("파일 변경됨 -> " + 
                                                         path.getFileName() + "\n"));
              } else if(kind == StandardWatchEventKinds.OVERFLOW) {
              }
            }
            boolean valid = watchKey.reset();
            if(!valid) { break;}
          }
        } catch (Exception e) {}
      }
    }
    
    TextArea textArea;
    
    @Override
    public void start(Stage primaryStage) throws Exception {
      BorderPane root = new BorderPane();
      root.setPrefSize(500, 300);
      
      textAra = new TextArea();
      textArea.setEditable(false);
      root.setCenter(textArea);
      
      Scene scene = new Scene(root);
      primaryStage.setScene(scene);
      primaryStage.setTitle("WatchServiceExample");
      primaryStage.show();
      
      WatchServiceThread wst = new WatchServiceThread();
      wst.start();
    }
    
    public static void main(String[] args) {
      launch(args);
    }
  }
  ```
